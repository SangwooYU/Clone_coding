  ######  css 관련
이미지 위치 포지션닝은
css - position - (static | fixed | relative | absolute)
static = default 
fixed - 사용자 브라우저를 기준으로 이동해도 같은 위치에 존재함.
like 스크롤을 내려도 같은 곳에 있는 광고들
relative - 기본적으로 표시된 위치를 기준으로 새 위치를 지정하는 것
absolute - 해당 요소의 위치가 지정된 부모 요소를 기준으로 fixed 시킨 것.
